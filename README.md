Project ini bertujuan untuk membedah fenomena "Tech Winter" melalui lensa data. Menggunakan dataset layoff global dari Kaggle, saya melakukan investigasi mendalam untuk menjawab apakah gelombang PHK massal ini merupakan tanda keruntuhan industri teknologi atau sekadar fase koreksi pasar.

Pos A (Sectoral Impact): Industri mana yang paling terpukul?

Pos B (Timeline Anatomy): Kapan puncak badai PHK terjadi?

Pos C (Company Maturity): Benarkah startup lebih rentan dibanding raksasa korporasi (Post-IPO)?
